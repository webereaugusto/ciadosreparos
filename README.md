# Cia dos Reparos - Site Institucional

Site institucional one page para a Cia dos Reparos, uma loja de materiais de construção, elétrica e encanamento localizada em Campinas, SP.

## Características

- Design responsivo
- Animações suaves
- Menu mobile
- SEO otimizado
- Carregamento rápido
- Compatível com todos os navegadores modernos

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- Font Awesome (ícones)
- Google Maps (localização)

## Estrutura do Projeto

```
.
├── index.html          # Página principal
├── styles.css          # Estilos do site
├── script.js           # Interações e animações
└── README.md          # Documentação
```

## Como Usar

1. Clone este repositório
2. Abra o arquivo `index.html` em seu navegador
3. Para desenvolvimento, utilize um servidor local

## Personalização

- As imagens podem ser substituídas no diretório `images/`
- As cores podem ser alteradas no arquivo `styles.css`
- Os textos podem ser editados no arquivo `index.html`

## SEO

O site foi otimizado para mecanismos de busca com:

- Meta tags apropriadas
- Estrutura semântica HTML
- Conteúdo relevante
- URLs amigáveis
- Imagens com alt text
